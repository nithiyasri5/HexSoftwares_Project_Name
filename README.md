Loan Eligibility Prediction with Approval Probability Scoring

Project Overview
This project aims to predict loan approval status using Machine Learning techniques. In addition to binary classification (Approved / Not Approved), the model also generates an Approval Probability Score to support better business decision-making.

Objectives
- Predict loan approval status
- Generate probability score for each applicant
- Analyze feature importance
- Provide business insights based on model output

Dataset Information
The dataset includes applicant details such as:
- Gender
- Marital Status
- Dependents
- Education
- Self Employment Status
- Applicant Income
- Coapplicant Income
- Loan Amount
- Loan Amount Term
- Credit History
- Property Area

Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

Model Workflow
1. Data Preprocessing
2. Handling Missing Values
3. Encoding Categorical Features
4. Train-Test Split
5. Model Training (Random Forest Classifier)
6. Model Evaluation
7. Feature Importance Analysis
8. Approval Probability Score Generation

Key Insights
- Credit History has the highest impact on loan approval decisions.
- Applicant Income and Loan Amount significantly influence approval probability.
- Applicants with strong credit history and stable income have higher chances of approval.

Outcome
The system successfully predicts loan approval status and provides interpretability through feature importance and probability scoring, making it useful for financial decision support systems.
